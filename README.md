# PilotAI

PilotAI is a lightweight small language model (SLM) forked from [Phi-1](https://github.com/microsoft/phi-1) under the MIT License.  
It is designed for fast, local inference and easy integration into operating systems, applications, and experimental hardware projects.

---

## ✨ Features
- **Compact design** — optimized for speed and low resource usage.
- **MIT Licensed** — free to modify, redistribute, and integrate.
- **Domain-tunable** — can be fine-tuned for OS tasks, game mechanics, or wearable devices.
- **Local-first** — runs without external API dependencies.
- **Flexible deployment** — supports GGUF, ONNX, and PyTorch runtimes.

---

## Get started
To get started, just visit the [PilotAI](https://echoai.joycode.co.uk), select PilotAI (or Gemini 2.5 Flash, but to use the official PilotAI, select PilotAI), and start chatting!
